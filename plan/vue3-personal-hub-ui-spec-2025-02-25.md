# 个人中心 / 作品集 — 首版 UI 与信息结构方案

**产出时间**: 2025-02-25  
**用途**: 作为「生成一版」的输入，供 MCP/设计稿生成与后续开发对照。

---

## 1. 产品定位与用户

- **主用户**：自己（收藏、工具、展示）
- **访客**：快速了解你、使用小工具
- **形态**：单页应用，部署 GitHub Pages，纯前端

---

## 2. 信息架构与路由

| 路由 | 页面 | 核心内容 |
|------|------|----------|
| `/` | 首页 | 站点标题、一句话介绍、四大入口卡片（学习收藏 / 工具集 / 项目展示 / 设置或关于） |
| `/bookmarks` | 学习收藏 | 左侧文件夹树 + 右侧列表（可筛选/搜索）；支持导入导出、浏览器书签导入 |
| `/tools` | 工具列表 | 工具卡片网格（图标 + 名称 + 简短描述），点击进入具体工具 |
| `/tools/:toolId` | 具体工具 | 如图片压缩：上传区 + 参数 + 下载；其他工具同理 |
| `/projects` | 项目展示 | 项目卡片列表（封面、标题、技术栈标签、时间） |
| `/projects/:id` | 项目详情 | 标题、描述、技术栈、链接、时间、MD 正文（难点/亮点） |

---

## 3. 布局与导航

- **桌面**：顶部导航栏（Logo/站点名 + 首页 / 学习收藏 / 工具 / 项目），右侧可放「语言切换」「主题切换」。
- **移动**：顶部导航 + 汉堡菜单折叠为抽屉，或底部 Tab 导航（首页 / 收藏 / 工具 / 项目）。
- **全局**：主内容区最大宽度可限制（如 1200px）居中，留白舒适。

---

## 4. 视觉风格（首版）

- **整体**：简洁、偏专业/作品集气质，避免花哨。
- **配色**：
  - 亮色：浅灰背景（如 #f5f5f5）、深灰正文（#333）、主色 1 个（如蓝 #2563eb 或品牌色）用于链接、按钮、选中态。
  - 暗色：深灰背景（#1a1a1a）、浅灰正文（#e5e5e5），主色适当提高亮度。
- **字体**：
  - 中文：系统无衬线（如 -apple-system, "PingFang SC", "Microsoft YaHei"）或 1 款 Web 字体。
  - 英文/数字：同上或 Inter / Roboto；标题可略加字重。
- **组件风格**：卡片带轻阴影、圆角（如 8px）；按钮主色填充或描边；表单输入框带清晰边框与 focus 环。
- **主题**：亮色/暗色切换，CSS 变量统一管理；可选跟随系统。

---

## 5. 中英文与可访问性

- **中英文切换**：导航或顶栏切换，持久化到 localStorage，可选 URL 参数。
- **可访问性**：语义化 HTML、焦点管理、键盘导航、关键交互 aria-label、图片 alt。

---

## 6. 关键页面 Frame 要点（供生成一版）

1. **首页**：顶部 Nav + 大标题 + 副标题 + 4 个入口卡片（等分或 2x2）。
2. **学习收藏**：顶 Nav + 左侧文件夹列表（窄栏）+ 右侧收藏列表（表头：标题/类型/标签/操作）。
3. **工具列表**：顶 Nav + 工具卡片网格（每卡片：图标、名称、一句话描述）。
4. **项目列表**：顶 Nav + 项目卡片网格（封面图、标题、技术栈标签、时间）。

以上作为「首版想法」输出，用于调用 MCP 或工具生成一版可看的 frame 设计稿。

---

## 7. frontend-design 规范（美学方向）

遵循 [anthropics/skills frontend-design](https://github.com/anthropics/skills)：避免通用 AI 审美，选定明确方向并执行到位。

| 维度 | 选择 | 说明 |
|------|------|------|
| **Tone** | 克制极简 + 编辑感 (refined minimal / editorial) | 个人中心/作品集：专业、可读、有记忆点，不花哨 |
| **Typography** | 标题：Syne（有性格的无衬线）；正文：DM Sans；中文：Noto Sans SC | 避免 Inter / Roboto / Arial；一对「展示 + 正文」搭配 |
| **Color** | 主背景：暖灰/米白；主色/强调：深墨绿 (#0f766e) + 琥珀高亮 (#d97706) | 主色主导 + 少量锐利强调，CSS 变量 |
| **Motion** | 首屏卡片 stagger 进入；卡片 hover 轻微上浮 + 阴影；焦点环过渡 | 高冲击时刻集中，CSS-only |
| **Spatial** | 主内容区略不对称（标题左对齐或偏左）；卡片网格留足负空间；可选一卡片略大或叠压 | 打破完全对称，留白充足 |
| **Background** | 浅色：微渐变 + 极细噪点纹理；深色：深底 + 微光 | 有层次与氛围，非纯色底 |
| **Differentiation** | 墨绿 + 琥珀的配色与 Syne 标题，在作品集类站点中易被记住 | 一眼可辨的视觉语言 |

实现时：与美学复杂度匹配；极简方向注重间距、字重与细节精度。
